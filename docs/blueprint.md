# **App Name**: WorkWise

## Core Features:

- User Authentication: User authentication via Google account or email/password combination.
- Daily Work Log: Daily work log input with date and text description fields.
- Log Tagging: Tagging system for work log entries (e.g., 'merged PR,' 'meeting,' 'debugging').
- Review Response Tool: AI-powered tool that checks user's logs from a certain period and generate responses based on what question the user inserts
- User Dashboard: Personalized dashboard displaying user's work log data.
- Data Visualization: Visually represent progress via charts
- Dark Mode: Switchable dark mode

## Style Guidelines:

- Primary color: HSL values of 215, 70%, 50%, resulting in a hex value of #3283D6, will be used to promote professionalism.
- Background color: A very light tint of the primary hue at HSL 215, 15%, 95% (Hex: #F0F4F9), suitable for a light color scheme
- Accent color: Analogous to the primary at HSL 185, 60%, 40% (Hex: #3FA6A6), chosen to be of adequate saturation to pop against the background and the primary, as the app will focus on the professional side.
- Headline font: 'Space Grotesk' (sans-serif) for headlines; body font: 'Inter' (sans-serif) for body text.
- Consistent set of professional icons. Icons to align with shadcn theme.
- Clean and structured layout using Shadcn components for a professional look. Prioritize data clarity on the dashboard.
- Utilize Shadcn components for a professional and customizable look and feel.
- Subtle transitions and animations for a polished user experience.
